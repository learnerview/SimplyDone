SimplyDone

A lightweight, generic, open-source background job scheduling engine built for developers.

Login / Get Started View on GitHub API Documentation

Open Source

Completely free and open-source. Host it yourself and integrate seamlessly into your microservices.

Webhook Driven

SimplyDone triggers HTTP callbacks (webhooks) to your execution endpoints. We've even added HMAC security to verify requests.

Reliable Queues

Configurable retries, priority queues, and Dead Letter Queues (DLQ) guarantee your background tasks won't get lost.
